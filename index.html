📆 Calendar Integration

Export your Clario 3.0 events and task due dates as an ICS file that can be imported into Outlook, Google Calendar, or Apple Calendar. Re-import Outlook calendars to sync back.

Auto-sync note: True 2-way Outlook sync requires the Outlook desktop add-in or Microsoft Graph API (OAuth). For now, export ICS from here and import manually in Outlook — takes under 30 seconds.
☁️ Google Drive Sync
—
Your data automatically syncs to your private Google Drive app folder whenever you make changes. Access your workspace from any device just by signing in.
Data is stored in a hidden app folder in your Google Drive — only Clario 3.0 can access it.
📱 Install as App

Clario 3.0 is a Progressive Web App (PWA). Install it as a native-feeling app on any device — no app store needed.

🪟 Windows / Chrome
Click the install icon (⊕) in the address bar, or go to Chrome menu → Cast, Save and Share → Install page as app
📱 iPhone / Safari
Tap the Share button (□↑) → Add to Home Screen → Add
🤖 Android / Chrome
Tap Chrome menu (⋮) → Add to Home screen or look for the install banner at the bottom
💻 Mac / Safari
File menu → Add to Dock (Safari 17+) or use Chrome/Edge with install icon
